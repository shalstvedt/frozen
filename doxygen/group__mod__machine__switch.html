request/switch
Machines

Switch module can pass requests to different machines according rules for incoming requests

Configuration

Accepted configuration: 

        {
                class      = "switch",
                rules      = {
                    { machine = (machine_t)'some1', request = { action = (action_t)'create'                    } },
                    { machine = (machine_t)'some2', request = { action = (action_t)'read', debug = (uint_t)'1' } },
                    { request = { test = (uint_t)'1' } }
                    { machine = (machine_t)'default' },
                }
        }

This config have 4 rules:

  • call "some1" machine on all ACTION_CREATE requests
  • call "some2" on ACTION_READ requests with debug == '1'
  • pass request to underlying machine then test == '1'
  • call "default" machine on any non matched request Last action is always - pass to underlying machine
