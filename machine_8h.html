API machine_t* shop_new ( hash_t *  config)

Create new shop.

Configuration is hash with items in reverse order. For example: 

       hash_t example_cfg[] = {
             { 0, DATA_HASHT(
                   { HK(class), DATA_STRING("storage/file") },
                   hash_end
             )},
             { 0, DATA_HASHT(
                   { HK(class), DATA_STRING("request/debug") },
                   hash_end
             )},
             hash_end
       };

This config produce two machines: debug and file. Debug will have one child - file and will pass all requests to it. machine_new will return pointer to last created machine. In this case - this is debug.

  • Use hash_null to avoid linkage between machine.
    Return values:
    NULLCreation failed
    non-NULLCreation success
