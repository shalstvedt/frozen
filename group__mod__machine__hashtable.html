index/hashtable
Machines

Description

This is simpliest index avaliable. For given value (i.e. from data hasher machine) it calculate offset in fixed size table and pass request. No shoping, nor collision detection. It more alike bloom filter, but for bigger than bit values.

Configuration

Accepted configuration: 

 {
              class                   = "index/hashtable",
              input                   = (hashkey_t)'keyid', # key for input, default "key"
              nelements               = (uint_t)'100',      # size of hash table
 }
