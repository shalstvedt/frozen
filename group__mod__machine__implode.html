request/implode
Machines

Description

This machine pack current request into one key-value pair in new request for underlying machine. Useful for ipcs, or for request storage. See shop/try for unpacking.This machine actually do nothing with request, just wrapping. Underlying machine must do all job with converting to packed state. "communication/ipc" already does it.

Configuration

Accepted configuration: 

 {
              class                   = "request/implode",
              buffer                  = (hashkey_t)'buffer',  # key name in new request
 }
