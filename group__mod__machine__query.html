Configuration

Accepted configuration: 

 {
              class                   = "data/query",
              data                    =                        # data to query
                                        (env_t)"data",         # - query data supplied in request
                                        (sometype_t)"",        # - custom data
                                        ....
              request                 = { ... }                # request for data query, optional, default is incoming request
 }
