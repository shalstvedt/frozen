bind_t
Data

Overview

This data is wrapper - it works with master data in runtime, and dump info to slave on destroy action. On creation data from slave transfered to master data.

Define

Possible defines in C code: 

       data_t bind = DATA_BINDT(_master,_slave,_format,_fatal);

In configuration file: 

  {
        master        = (some_t)"",         // master data, would be used in runtime
        slave         = (some_t)"",         // slave data, used to save and load information
        format        = (format_t)"clean",  // format in which data would be stored, default FORMAT(binary)
        fatal         = (uint_t)"1"         // do not ignore load errors, default 0
  }
  • Master data should support convert_to and convert_from with given format
  • Slave should support basic read/write api, and, obviously, to write information on disk or on some storage device
