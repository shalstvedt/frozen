named_t
Data

Overview

This data assign names for specified data for further usage.

Define

Possible defines: 

       data_t hnamed = DATA_NAMEDT(data);

Behavior

  • COPY: because of global nature of named_t data, not possible to have copies of one data under same name, so COPY action not really performed - we only increment internal ref counter for that, and return same data for every copy.
  • CONVERT_FROM: then converting from hash supplied data copied and used in further actions.
