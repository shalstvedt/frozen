How to install

Get archive

To get archive you can use any of the following methods:

  • Download as .zip
  • Download as .tar.gz
  • git clone https://github.com/x86-64/frozen.git
  • git clone git://github.com/x86-64/frozen.git

Compile

Compilation as easy as any other package.

1
2
3
4
$ ./configure
$ make
$ make check
$ make install

"make check" is optional, but recommended.

Configuration script automatically find all installed software, such as ZeroMQ library and use it. If not - this modules would be disabled and installation continue.
